# ChatSpace
a chat app demo

## info

developing with ``koa``, ``vue`` and ``websocket``

## install

> git clone https://github.com/the-Order-of-the-Phoenix/ChatSpace.git
> npm i
> npm start

## dependency

- node >= 8.0
- pm2

## run

> npm start

its' running in ``html://localhost:3000/index.html``
